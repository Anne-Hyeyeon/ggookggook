# ggookggook App Structure

## User Flow

```
메인 (인체 SVG)
       │
       ├── 부위 클릭 ──→ 부위 확대 뷰 ──→ 혈자리 점 표시 ──┐
       │                                                    │
       ├── 증상 선택 ──→ 인체에 혈자리 하이라이트 ──────────┤
       │                                                    │
       └── 검색 ──────→ 검색 결과 목록 ──→ 혈자리 선택 ────┘
                                                            │
                                                            ▼
                                                   혈자리 상세 패널
                                              (이름, 효능, 지압법, 유튜브 추후)
```

## Pages

| Path | Page | Description |
|------|------|-------------|
| `/` | 메인 | 인체 SVG + 증상 탭 + 검색바 |
| `/favorites` | 즐겨찾기 | 저장한 혈자리 목록 |
| `/login` | 로그인 | Supabase Auth |
